# SpringBoot酒店管理系统 java1210

## 项目概述

本系统是基于前后端分离的酒店管理系统，后端采用SpringBoot框架，结合Mybatis持久层框架与MySQL数据库；前端则使用Vue.js进行构建。

## 技术栈

- 后端：SpringBoot、Mybatis、MySQL
- 前端：Vue.js
- 推荐开发环境：IDEA、JDK 1.8、Maven、MySQL

## 功能模块

### 系统角色

- 管理员
- 用户端

#### 管理员功能

- 用户管理：查询用户、修改用户信息、删除用户
- 订单管理：处理等待列表、取消订单、接收订单
- 房间管理：管理房间列表、查询房间详情、添加房间、修改房间信息、删除房间
- 评论管理：查看用户评论

#### 用户端功能

- “我的”：查看我的订单、个人信息、修改密码
- 订房间与找房间
- 查询空余房间

- 后端服务访问地址：[http://localhost:9152/](http://localhost:9152/)
- 前端服务访问地址：[http://localhost:9153/](http://localhost:9153/)

## 部署步骤

1. 环境准备：确保IDEA、JDK 1.8、Maven、MySQL已正确安装与配置。
2. 数据库配置：根据项目需求初始化数据库。
3. 后端部署：通过Maven构建项目，并启动SpringBoot服务。
4. 前端部署：在准备好Vue.js环境后，构建并部署前端项目。

## 核心亮点

- 基于前后端分离的架构，有利于前后端分工合作，提高开发效率。
- 使用SpringBoot框架，简化了传统的Spring应用开发流程，实现快速开发。
